Setup

Install and configure Ralph CLI

ralph install

Beginner

Install Ralph templates and configuration into your project

Syntax
Terminal 
ralph install [--skills]
Flags
--skills Install optional skills (commit, dev-browser, prd)
Examples
Terminal 
# Basic installation
cd my-project && ralph install

# With optional skills
ralph install --skills
See also: ralph prd ralph plan

PRD (Product Requirements)

Create and manage product requirements documents

ralph prd

Beginner

Generate a Product Requirements Document interactively

Syntax
Terminal 
ralph prd
Usage

Opens an interactive session where you describe your feature. Ralph will generate a structured PRD with user stories and acceptance criteria.

Terminal 
# Creates .ralph/PRD-N/prd.md
ralph prd
Output

Creates a new folder .ralph/PRD-N/prd.md where N auto-increments (PRD-1, PRD-2, etc.)

See also: ralph plan ralph stream new

Planning

Create implementation plans from your PRD

ralph plan

Beginner

Generate an implementation plan from your PRD

Syntax
Terminal 
ralph plan [--prd <path>] [--plan <path>]
Flags
--prd <path> Path to custom PRD file (default: auto-detected)
--plan <path> Path to output plan file
Examples
Terminal 
# Plan the latest PRD
ralph plan

# Plan a specific PRD
ralph plan --prd docs/prd-api.md --plan .ralph/api-plan.md
Output

Creates .ralph/PRD-N/plan.md with tasks for each user story, including scope, acceptance criteria, and verification commands.

See also: ralph prd ralph build

Building

Execute your plan and build features

ralph build

Beginner

Run build iterations to implement your plan

Syntax
Terminal 
ralph build <iterations> [options]
Flags
<iterations> Number of stories to process (required)
--no-commit Dry run without committing changes
--prd <N> Specify which PRD to build (e.g., --prd=1)
--agent <name> Override agent (claude, codex, droid)
--plan <path> Path to custom plan file
--progress <path> Path to custom progress file
Examples
Terminal 
# Run 5 iterations
ralph build 5

# Dry run (no commits)
ralph build 1 --no-commit

# Build specific PRD
ralph build 5 --prd=1

# Use a different agent
ralph build 3 --agent=codex
See also: ralph plan ralph stream build

Streams (Parallel Execution)

Work on multiple features simultaneously using git worktrees

ralph stream new

Intermediate

Create a new PRD folder for parallel work

Syntax
Terminal 
ralph stream new
Output

Creates a new .ralph/PRD-N/ folder where N is the next available number.

See also: ralph stream init ralph stream list

ralph stream list

Beginner

List all streams with their current status

Syntax
Terminal 
ralph stream list
See also: ralph stream status

ralph stream status

Beginner

Show detailed status table for all streams

Syntax
Terminal 
ralph stream status
Output

Displays a table showing each stream's status (ready, running, completed, merged), progress, and worktree state.

See also: ralph stream list

ralph stream init

Intermediate

Create an isolated git worktree for a stream

Syntax
Terminal 
ralph stream init <N>
Examples
Terminal 
# Initialize worktree for PRD-1
ralph stream init 1

# Initialize worktree for PRD-2
ralph stream init 2
Output

Creates a git worktree at .ralph/worktrees/PRD-N/ on a new branch ralph/PRD-N.

See also: ralph stream build ralph stream cleanup

ralph stream build

Intermediate

Run build iterations in a stream's worktree

Syntax
Terminal 
ralph stream build <N> <iterations> [options]
Flags
<N> Stream/PRD number (required)
<iterations> Number of stories to process (required)
--no-worktree Build directly without using worktree
--force Force build even if lock exists
Examples
Terminal 
# Build PRD-1 with 5 iterations
ralph stream build 1 5

# Build without worktree
ralph stream build 1 5 --no-worktree

# Run two streams in parallel
ralph stream build 1 5 &
ralph stream build 2 5 &
wait
See also: ralph stream init ralph stream merge ralph build

ralph stream merge

Intermediate

Merge a completed stream back to main branch

Syntax
Terminal 
ralph stream merge <N>
Examples
Terminal 
# Merge PRD-1 to main
ralph stream merge 1
See also: ralph stream build ralph stream cleanup

ralph stream cleanup

Intermediate

Remove a stream's worktree after merging

Syntax
Terminal 
ralph stream cleanup <N>
Examples
Terminal 
# Clean up after merging
ralph stream cleanup 1
Output

Removes the worktree at .ralph/worktrees/PRD-N/. The PRD files remain in .ralph/PRD-N/.

See also: ralph stream merge

ralph stream mark-completed

Advanced

Manually mark a stream as completed (for direct-to-main builds)

Syntax
Terminal 
ralph stream mark-completed <N>
See also: ralph stream verify-status

ralph stream verify-status

Advanced

Auto-scan and fix stale status markers

Syntax
Terminal 
ralph stream verify-status
Output

Checks git history against file markers and auto-creates missing .completed or .merged markers when commits exist.

See also: ralph stream status ralph stream mark-completed
